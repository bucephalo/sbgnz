Aplicación móvil (Angular + Ionic) App móvil para apoderados (en desarrollo)
Aplicación para apoderados con hijos escolares. La app informa sobre diferentes asuntos relacionados con el colegio y las clases (notas, asistencia, promedios, alertas, etc.)
En la web app (también en desarrollo), los profesores o encargados ingresan la información que se desplegará en la aplicación móvil.
Utilizamos Angular + Ionic + Scss.
Aplicación web (Angular + Bootstrap) App web para profesores (en desarrollo)
Sistema web para profesores y/o encargados de ingresar la información que se desplegará en la aplicación móvil; y también para uso personal. Permite "cargar" alumnos desde archivos excel, crear asignaturas, planes de estudio, ingresar notas, entre otros.
Utilizamos Angular + Bootstrap.
Sistema web/móvil (Vue + Tailwindcss) Agenda web responsiva (en desarrollo)
Los pacientes de Red Interclínicas podrán agendar horas con los profesionales a través de la web o sus teléfonos.
